## Requirements

* [Node.js][node] 14+



## Usage

### Initial setup

```bash
# Clone the repository.
git clone https://github.com/MediaComem/rps.git

# Install dependencies.
cd rps
npm ci
```

### Development

```bash
# Run the application in development mode with live code reload.
npm run dev
```

### Production

```bash
# Run the application in production mode.
npm start
```
